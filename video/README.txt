Place your surprise video here!

1. Add your video file and name it: surprise.mp4
   (The video player is configured to play this file)

2. Supported format: MP4 (recommended)
   Other formats: webm, ogg (update the HTML source tag)

3. To use a different filename:
   - Edit index.html
   - Find the <video> tag
   - Update: <source src="video/YOUR-FILENAME.mp4" type="video/mp4">

4. Recommended video specs:
   - Resolution: 1920x1080 or lower
   - Format: H.264 MP4
   - Keep file size reasonable for web playback
